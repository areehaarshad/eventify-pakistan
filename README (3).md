# Eventify Pakistan

A high-fidelity, interactive front-end prototype of an event-discovery platform for Pakistan — designed in Figma and built as a working clickable website.

**Live demo:** _(add your GitHub Pages link here after deployment)_

## About the Project
Eventify Pakistan helps users discover events happening across the country — from tech conferences and university convocations to trade exhibitions — all in one place. This build turns the original Figma design into a fully navigable prototype.

## Screens Included
- **Sign Up / Login** — account creation with city selection
- **Home** — featured events and upcoming event plans
- **For You** — personalized event recommendations by category
- **Following** — events from organizations the user follows
- **Blog** — featured news article layout (UMT 27th Convocation)
- **About Us** — mission statement and brand story
- **Profile** — user bio, attended events, personality tags, badges
- **Settings** — account preferences, security, visibility, notifications

## How the Flow Works
Click through the top navigation (**Home / For you / Following**), the settings gear icon, or the footer links (**Blog / About Us / Profile / Settings**) to move between screens — just like the original Figma prototype flow.

## Tech Used
- HTML5, CSS3, vanilla JavaScript (no frameworks — fast load, easy to deploy)
- Fully responsive layout
